## Cài đặt

Đầu tiên, tải repository về máy tính:

```bash
$ git clone 
```

Cài đặt requirements:

```bash
$ cd myproject-django
$ source env/bin/activate
```

Tạo database:

```bash
$ python manage.py migrate
```
Chạy development server

```bash
$ python manage.py runserver
```

Mở Chrome hay FireFox : **127.0.0.1:8000**
